# U Devs Management System (Frontend)

##  Project Overview
U Devs Management System is a **fully responsive frontend website** designed to manage users, projects, and services efficiently.  
This project demonstrates **modern UI/UX**, Bootstrap-based layout, and professional design for smooth workflow.

The website includes:
 Navbar with logo and navigation links
 Hero section with a responsive slider
  Features / Services cards
 About section with image and info
 Contact form with basic JavaScript validation
 Login modal popup
 Footer

The project is built using **HTML, CSS, and Bootstrap 5**.
